# 笔记——成长

以下为一些阅读体会和总结。



## 《跃迁：成为高手的技术》 —— 古典

1st

- 前面的节奏是 “时代什么在变” - “怎么抓机会” - "如何学习"。
  - 对于跃迁，这是一个 停滞 - 突飞 - 停滞 - 突飞 的过程
  - 此时你需要借助大势来提升自己，利用身边人的智慧，自己只做最主要、最精华的一部分。
- 要在高价值领域持续做正确动作。
  - 抓住最有可能的那个机会，可以放弃其他把握不大的机会。同时要使用三次跃迁来完成蜕变。
  - 首先用二八法则高速成长，先投入20%的时间到80%的工作里面；
  - 然后利用系统扩大名气，利用你比别人多的东西与平台合作，获得更多资源来提升自己；
  - 最后要和优秀的大脑互联，多和聪明人聊天。
  - 综合来说，要对内使用二八法则³不断扩大自我效能，对外移动到系统头部（越往前资源越多，指数增长）获得系统推力。
  - 在进行实践时，专注一件事，不断迭代，不指望妙手，下好51%效率的本手即可。本手可复刻，妙手不可复刻。

> ​	高价值：期望获利，即E(x)，最高的东西

- 要联机学习。
  - 首先站到**知识源头**，辨别一手知识，走向知识源头，和源头站在一起
  
  > ​	一手知识：研究论文，行业的学术期刊，行业最新数据报告，研究论文，行业的学术期刊，行业最新数据报告；
  >
  > ​	二手知识：名校的教科书、MOOC里推荐的一手材料、维基百科、中立第三方的行业调查报告、讲述底层逻辑、思考质量比较高、略微难懂的书和文章，比如《国富论》《穷查理宝典》《决策与判断》等；
  >
  > ​	三手知识：畅销书等方便传播的内容，其在原基础上加入大量案例、故事以及不精确的概念。
  >
  > ​	四手知识：根据三手知识和个人体验生成的鸡汤，故事量 > 知识量
  
  - 然后极其**功利地读书**，读书需要功利性地读书，**有目标导向、有即时反馈、最近发展区**，有规划地将资源配置在心智类（爬坡）、知识类（平地）、娱乐类（下坡）中，设置不同的目标。读书也不要从第一页开始读，直接读你需要的内容。
  - 最后凝聚一颗**知识晶体**，让知识点之间形成稳定架构，如树状、关联、序列、数据。
  - 进行联机学习时，你要借助身边人的力量，理解知识晶体本质上不是在交换，而是在增长。
  - 和知识源头进行联机，和优秀的、有见解的人进行联机，和跨行业的高手进行联机。
  - 多问有意义的问题，因为学是学不完的，但是遇到的问题是能够被不断解决的，因此要以问题为导向地进行学习。可以罗列问题清单，以 “问题” - “相关性” - “好奇心” 的方式进行评估，相关性是完成问题后立刻获得收益的内容，好奇心是你最有学习动力的方向。
  - 做好一个“问题少年”。提前准备内容，不伸手要饭；问问题可以不断深入，可能别人也可以借助这个机会整合自己的知识；一定要输出答案，不然白问。可以从证据、视角、联系、猜想、相关五个视角提问。
  - 不憋大招，慢慢输出。敢于分享，错了就改。放大碎片化的价值，设计属于自己的MVP（最小可交付产品）
  - **用提问学习，用联机思考，用输出整合**
  
- 要有破局思维

  - 当发现自己进行轮回，寻找方法跳出循环。此时积极升维，不纠结这个事情，而是高维看本质，寻找新方法新思路
  - 系统看问题。关注系统中元素之间的关系而不是元素本身。
  - 学习一些自增长技能。读写能力、破英语、社交能力、解决问题、聪明的善良。
  - 更少目标，战略勤奋 > 方法论勤奋 > 努力。相信自己遇到的问题前人都经历过，并有一套很好的方法论，第一步应该是学习。想明白做事的目标，了解其中的关系才能少做事，找到杠杆的支点。努力的收益不高，就要找方法；方法论学不过来，就要重新设定目标。
  - 脱困四问。“我在什么情绪之中？给自己的情绪打几分？”、“发生了什么？客观评价 otherwise回去评估情绪”、“我原本想要什么？找到期望目标差距”、“我如何改进？”
  - 做事情要多层布点，越多的途经点才能让自己越精确地到达目标。途经点不一定是直觉上要做的事情，两点之间，不是直接最快，而是阻力最小的那条线最快。
  - 要做善良的人。善待亲人，和梦想一致的战友、成长速度一样的伙伴、支持情感的朋友深交，持续感谢有实力且帮助过你的人。要做足够大的梦，快速成长让别人看到。
  - 要变得可激怒，不怕事，直接报复回去。要简单，做事不弯弯绕绕，做人不遮遮掩掩，这样别人才和你玩。





  ## 有价值的事情
  #### 1、思想夜宴
  带上你熟人认识的大牛们（关系不亲切，能力强，领域宽泛）一起，每个人准备8分钟干货（最好是系统性的知识和小tips，不谈个人经历或者感悟），接受追问。然后休息15分钟（大家私下聊天），休息后进行提问，每个人说出自己希望一起讨论的问题，大家投票给最有价值或者自己最感兴趣的问题，大家聊这个问题（每人3min）
